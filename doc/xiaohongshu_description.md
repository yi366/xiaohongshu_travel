# 小红书文旅笔记爬虫项目说明
## 一、项目简介
本项目基于Python实现小红书文旅类笔记批量爬取，采集旅游攻略类笔记的标题、正文、点赞数、收藏数、笔记链接，将采集到的文旅数据集导出为CSV文件，用于文旅内容数据分析。

## 二、项目目录结构
- code：爬虫主程序 spider.py
- data：存放爬取后生成的数据集 xhs_data.csv
- doc：项目说明、实验报告文档
- README.md：仓库总介绍

## 三、所需依赖库
pip install requests pandas beautifulsoup4

## 四、主要使用步骤
1. 小红书搜索文旅相关笔记（城市旅游、Citywalk、小众景点等），复制带/item/的笔记详情链接
2. 将多条链接填入代码中 url_list 列表
3. VSCode终端运行爬虫：
python code/spider.py
4. 运行完成后，data文件夹自动生成旅游数据集csv文件

## 五、功能说明
1. 支持批量输入多条笔记链接一次性爬取
2. 自动存储结构化数据，方便后续统计分析
3. 内置随机延时，降低爬虫被限制风险

## 六、数据字段
标题、正文、点赞数、收藏数、笔记链接
