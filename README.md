# xiaohongshu_travel 小红书文旅爬虫项目
## 项目功能
1. 支持批量输入多条笔记链接一次性开展爬虫采集
2. 自动把采集结果存成结构化csv数据集，便于后续做数据分析统计
3. 代码内置随机延时，降低爬虫被平台限制封禁的概率
4. 采集字段：笔记标题、正文内容、点赞数、收藏数、原始笔记链接

## 数据集特殊说明
受小红书网页反爬机制限制，静态爬虫无法获取真实点赞、收藏互动数据，互动字段统一脱敏显示为0。
本项目采用话题标签数量划分爆款/普通笔记：
1. 爆款笔记：绑定3个及以上文旅类话题标签
2. 普通笔记：绑定2个及以内话题标签

配套 `data_classify.py` 代码可自动给数据集新增「笔记类型」分类字段，运行后生成 labeled_xhs.csv，供后续分组对比传播特征分析。
