一、项目数据来源
本数据集通过 Python 爬虫工具抓取小红书公开文旅类旅游笔记内容，仅采集平台对外公开展示的笔记文本信息，不获取用户手机号、私信、身份等私密隐私数据。
数据仅用于高校文旅内容数据分析课程小组作业，严禁商用、二次分发。
二、数据集文件介绍
项目data文件夹下共三份 CSV 数据文件：
1.raw.csv：爬虫初次抓取的原始未清洗数据集，存在正文内容重复、格式错乱问题；
2.xhs_data.csv：人工修正正文、清理乱码后的基础干净数据集；
3.labeled_xhs.csv：经过data_classify.py脚本自动标注分类后的数据集，区分爆款 / 普通笔记。
三、爆款与普通笔记划分标准
本次爬虫抓取的笔记点赞、收藏字段数值均为 0，无法依靠互动量划分流量层级，因此采用标题关键词判定法：
1.爆款笔记：标题包含「攻略、必去、封神、绝美、天花板、宝藏、绝美打卡」等流量热门词汇；
2.普通笔记：标题无引流热门关键词，仅简单记录日常出游。
四、数据集字段说明
每条笔记包含字段：笔记标题、正文内容、旅游地点、配图链接、发布时间。
五、数据采集伦理规范
1.仅爬取平台完全公开可见内容，不绕过平台限制访问隐藏、私密笔记；
2.不采集任何用户个人隐私信息，无用户 ID、联系方式等敏感数据；
3.爬虫设置合理访问间隔，不对小红书服务器造成高频压力；
4.数据集仅限课堂教学分析使用，不对外传播、不用于商业盈利。
六、数据处理流程
1.运行spider.py批量爬取文旅笔记，导出原始raw.csv；
2.人工修复正文乱码、重复内容，生成清洗文件xhs_data.csv；
3.执行data_classify.py，根据标题关键词自动给每条笔记打标签，输出分类完成的labeled_xhs.csv。
